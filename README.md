# Project Management Tool

## Description
This is a simple project management to-do list tool created using HTML, CSS, and JavaScript

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks from the list

## How to Run
1. Download or clone the repository
2. Open the index.html file in any web browser

## Additional Information
To mark a task as completed, you select the task itself. When you select it, the task will turn green to indicate it has been marked as completed. To undo it just click on the task again<br>
To delete a task from the list simply press the delete button to the right of the task itself
